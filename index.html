This website has threepages:
A main page that provides an overview of the project and includes links to the other 3 relevant pages;
Each containing a form that serves as a web interface for the web service, http://www.random.org/. These forms include:

  • An integer generation form that allows users to generate a specified number of integers within a given range
  • A sequence generation form that allows users to generate a specified number of items within a given range
  • String generation form that allows users to generate a specified number of strings of a certain length and characters
